üéì Practice Problems

Problem 1: Single Number

Given array where every element appears twice except one, find the single element.

Input: [4,1,2,1,2] ‚Üí Output: 4

Solution: XOR all numbers. Duplicate numbers cancel out!

result = nums[0] ^ nums[1] ^ nums[2] ^ ...

Problem 2: Number of 1 Bits

Count the number of '1' bits in a number (Hamming weight).

Input: 11 (1011) ‚Üí Output: 3

Solution: n & (n-1) removes rightmost 1

while(n) { count++; n = n & (n-1); }

Problem 3: Power of Two

Check if a number is a power of 2.

Input: 16 ‚Üí Output: true

Solution: Power of 2 has only one set bit

return n > 0 && (n & (n-1)) == 0

Problem 4: Reverse Bits

Reverse the bits of a 32-bit unsigned integer.

Input: 00000010100101000001111010011100

Solution: Extract each bit and build result from right

result = (result << 1) | (n & 1); n >>= 1;
