## üíª Usage

### Interactive Visualizer
1. Enter numbers or click individual bits to toggle
2. Select an operation mode (AND, OR, XOR, etc.)
3. Watch bits transform with color highlighting
4. Try 8 common tricks and solve practice problems

### Python Playground
1. Write bit manipulation code in the editor
2. Click "‚ñ∂ Run Code" or press `Ctrl+Enter`
3. View step-by-step visualization of each operation
4. Track variables with automatic binary display

**Example:**
```python
a = 12  # 1100
b = 10  # 1010
result = a & b  # 1000 = 8
print(f"Result: {result}")
```
